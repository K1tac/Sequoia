# Sequoia

Sequoia is a small language that compiles to C, builds the generated program with `gcc`, and immediately runs it. The current language is centered on integer math, functions, conditionals, `while` loops, console I/O, and small interactive programs.

Start with the local language spec in [LANGUAGE_SPEC.md](LANGUAGE_SPEC.md), or browse the [project wiki](https://github.com/K1tac/Sequoia/wiki).
